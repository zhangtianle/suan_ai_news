# News Aggregator 项目

科技 & 财经资讯聚合与分析系统

## 🎯 项目定位

为投资者和科技从业者提供有价值的资讯分析：
- **科技资讯** - AI、芯片、互联网、创业投资
- **财经资讯** - 宏观政策、股市、商品、外汇
- **市场情绪** - 多空信号识别
- **投资决策** - 风险预警、机会挖掘

## 📁 项目结构

```
tech-news/
├── README.md
├── scripts/
│   ├── run_all.sh           # 统一运行脚本
│   ├── tech_crawler.py      # 科技资讯爬虫
│   ├── tech_processor.py    # 科技资讯处理
│   ├── tech_analyzer.py     # 科技资讯报告
│   ├── finance_crawler.py   # 财经资讯爬虫
│   ├── finance_processor.py # 财经资讯处理
│   └── finance_analyzer.py  # 财经分析报告
├── data/
│   ├── raw/                 # 科技资讯原始数据
│   ├── processed/           # 科技资讯分类数据
│   └── finance/             # 财经资讯数据
├── output/
│   ├── tech/                # 科技资讯报告
│   └── finance/             # 财经分析报告
├── logs/                    # 日志文件
└── sources/                 # 数据源配置
```
